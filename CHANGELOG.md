## [1.2.0] - Previous Release

### Added
- Persistent toolbar feature (FoxClocks-style)
- Country flags for timezones
- Custom timezone aliases
- Configurable toolbar position (top/bottom/left/right)
- Website blacklist functionality
- Toolbar minimize and hide controls

### Features
- ⏰ Multiple timezone clock display
- 🌐 Persistent toolbar on all pages
- 📍 Configurable positioning
- 🚫 Website blacklist support
- ✏️ Custom aliases for timezones
- 🏴 Country flag indicators
- 🔍 Timezone search functionality
- 🎨 Modern dark-mode UI
- ⚙️ Customizable time/date formats
- 🌍 50+ pre-configured cities
- 🔄 Automatic DST handling
- 💾 Cloud sync support

### Technical
- Manifest V3 compliance
- Support for Chrome, Firefox, and Edge
- Service worker-based background script
- Content scripts for toolbar injection
- Chrome storage API integration

## Version Types

- **Major** (X.0.0): Breaking changes or major feature additions
- **Minor** (1.X.0): New features, backward compatible
- **Patch** (1.2.X): Bug fixes and minor improvements
