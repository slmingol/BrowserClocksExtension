# Screenshots Guide

Please capture the following screenshots and save them in this folder:

## Required Screenshots

1. **popup.png** - The extension popup
   - Click the extension icon to open the popup
   - Make sure you have 3-5 clocks added with different timezones
   - Show the search/add functionality
   - Recommended size: 400x600px or larger

2. **toolbar.png** - The persistent toolbar on a webpage
   - Visit any website with the toolbar enabled
   - Show the toolbar with multiple clocks displaying
   - Show country flags visible
   - Capture enough context to show it's on a real webpage
   - Recommended size: 1200x200px or larger

3. **options.png** (optional) - The settings/options page
   - Open the extension settings
   - Show the configuration options
   - Recommended size: 800x600px or larger

## How to Take Screenshots

**macOS:**
- Full screen: `Cmd + Shift + 3`
- Selection: `Cmd + Shift + 4`
- Window: `Cmd + Shift + 4`, then press `Space`, click window

**Windows:**
- Snipping Tool or `Win + Shift + S`

**Chrome DevTools:**
- Right-click on extension popup → Inspect
- Use Chrome's screenshot tool for perfect sizing

## Naming Convention

- `popup.png` - Main popup interface
- `toolbar.png` - Toolbar on webpage
- `options.png` - Settings page
- `popup-dark.png` - Dark mode popup (if applicable)
- `toolbar-compact.png` - Compact view (if applicable)

After adding screenshots, they'll automatically appear in the README.
